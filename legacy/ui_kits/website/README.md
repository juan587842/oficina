# Website UI Kit — Fabio Mecânica Diesel

Marketing one-pager for the workshop. Conversion goals:
1. Call / WhatsApp for service
2. See services & trust signals
3. Get hours & location

## Components
- `Nav.jsx` — sticky top nav with logo + CTAs
- `Hero.jsx` — "Caminhão parado é prejuízo" headline + CTAs
- `ServicesGrid.jsx` — 6-card grid of services with icons
- `TrustBar.jsx` — years in business, trucks serviced, fleet clients
- `HoursMap.jsx` — hours table + placeholder map
- `ContactModal.jsx` — WhatsApp-style quick-contact form
- `Footer.jsx`
- `WhatsAppFloat.jsx` — fixed bottom-right
- `PrimaryCTA.jsx` — reusable CTA block

Open `index.html` for the full interactive mock.
